# Public Boundary

这个仓库可以公开，因为它已经把原项目中的敏感内容抽离。

## 保留

- ESP32-C3 + PlatformIO + Arduino 工程结构。
- NimBLE-Arduino 的 BLE 外设、服务、特征值、通知、扫描用法。
- 命令解析、UART 转发、状态日志。
- Xbox 控制器输入包解析思路。
- 原项目编译验证记录。

## 移除

- 真实设备名、业务名、客户名。
- 真实 MAC 地址。
- 原始工程本机路径、项目代号和目录名。
- 真实控制对象和私有业务协议。
- 内部调参值和交付上下文。
- 第三方 Xbox 连接头文件的原始复制件。

## 对外说法

可以说：

> 这是从真实硬件联调项目中提炼的公开版本，用于展示 ESP32-C3 BLE 控制链路、串口转发和控制器输入解析能力。原始项目因包含真实设备与业务协议，不直接公开。

不要说：

> 这个仓库就是原始交付项目。

它不是原始交付项目，而是作品集公开版本。
