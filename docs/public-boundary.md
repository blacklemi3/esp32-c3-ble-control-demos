# Public Boundary

这个仓库已经抽离原项目中的敏感内容，只保留通用代码结构和示例流程。

## 保留

- ESP32-C3 + PlatformIO + Arduino 工程结构。
- NimBLE-Arduino 的 BLE 外设、服务、特征值、通知、扫描用法。
- 命令解析、UART 转发、状态日志。
- Xbox 控制器输入包解析思路。
- 原项目编译验证记录。

## 移除

- 真实设备名、业务名、客户名。
- 真实 MAC 地址。
- 原始工程本机路径、项目代号和目录名。
- 真实控制对象和私有业务协议。
- 内部调参值和交付上下文。
- 第三方 Xbox 连接头文件的原始复制件。

## 公开说明

这是从真实硬件联调项目中整理出的公开版本，保留 ESP32-C3 BLE 控制链路、串口转发和控制器输入解析的通用部分。

原始项目包含真实设备与业务协议，因此不直接公开。这个仓库不是原始交付项目，而是经过脱敏和重写的公开示例。
